# Changelog

All notable changes to the Dunam Ai project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0-beta.1] - 2026-01-30

### Added
- **Real-time Notifications**: Integrated persistent system alerts for bot events (Start, Stop, Errors, Daily Limits).
- **Admin Announcements**: Backend system and frontend polling for global official announcements.
- **Bug Reporting**: Premium "Report a Bug" modal in the sidebar for user feedback during beta.
- **Version Tracking**: Formal tag `v1.0.0-beta.1` and `CHANGELOG.md`.

### Changed
- **Auth UI Branding**: Removed logos from login/signup pages and refined typography for a premium look.
- **Unified Versioning**: Synchronized backend and frontend version numbers.

### Fixed
- Notification bell dropdown stability and state management.
- Dashboard stats hydration logic.
